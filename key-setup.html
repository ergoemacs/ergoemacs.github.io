Supported Key Themes and Keyboard Layouts

To use key theme and layout, use the following in your .emacs or .emacs.d/init.el. 

(setq ergoemacs-theme nil)
                (setq ergoemacs-keyboard-layout "us")
                (ergoemacs-mode 1)
